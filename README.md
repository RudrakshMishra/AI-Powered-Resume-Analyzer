# AI-Powered Resume Analyzer  
Full-Stack AI Application | Resume Feedback & ATS Optimization

##  Overview

AI-Powered Resume Analyzer is a full-stack web application that helps job seekers instantly analyze and improve their resumes using Artificial Intelligence and Natural Language Processing (NLP).

Users can upload their resumes (PDF or DOCX), and the system extracts key information such as skills and experience, identifies weak points, and provides actionable improvement suggestions similar to modern ATS (Applicant Tracking System) tools used by recruiters.

This project demonstrates real-world AI integration, backend API development, file parsing, and a clean dashboard UI, making it ideal for portfolios, freelance work, and SaaS-based products.
